Como podrás ver en unos momentos todo esta organizado en carpetas y todo debería ser fácil para buscar las cosas.

CSS:
	Solo habrá carpetas de CSS y se separa de nuevo dependiendo la sección ya sea inicio, perfil, iniciar sesión, etc.

JS: 
	Solo carpetas de JS y se separa igual que por secciónes de la página.

HTML
	Solo va haber HTML y se separa por secciones
